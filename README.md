## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/IosebKoplatadze/geo-decision.git
```

Go to the project directory

```bash
  cd geo-decision
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
